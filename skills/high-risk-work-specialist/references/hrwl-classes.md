# Australian High Risk Work Licence (HRWL) classes

The national HRWL classes, set by **Schedule 3 of the model WHS Regulations** and issued by each
state/territory regulator (and WorkSafe Victoria). **Australia only** — New Zealand has no HRWL (see
`nz-competency.md`). **Not legal advice**; class codes change — treat as verify-pointers and confirm
with the regulator.

## How the system works

- A person must hold the **relevant HRWL class** to carry out that high-risk work (or be a trainee
  under direct supervision of a licence holder).
- Training and assessment must be done through a **Registered Training Organisation (RTO)**; the
  **licence is then issued by the regulator** in the state/territory.
- Licences are **nationally recognised** and valid for **5 years**.
- Licensing classes are **hierarchical** — a higher class generally covers the work of the lower
  classes in the same stream (e.g. advanced rigging covers basic; a higher-capacity slewing crane
  class covers lower capacities). Confirm the exact coverage with the regulator.

## Scaffolding

| Code | Class | Covers |
|---|---|---|
| **SB** | Basic scaffolding | Modular/prefabricated scaffolds, bracket scaffolds, cantilevered materials hoists (≤ 500 kg), safety nets, etc. |
| **SI** | Intermediate scaffolding | SB plus tube-and-coupler scaffolds, cantilevered/spurred scaffolds, mast-climbing platforms, etc. |
| **SA** | Advanced scaffolding | SI plus hung scaffolds and suspended scaffolds |

## Dogging and rigging

| Code | Class | Covers |
|---|---|---|
| **DG** | Dogging | Slinging techniques (selecting/inspecting lifting gear, balancing/securing loads) **and** directing a crane operator in moving a load |
| **RB** | Basic rigging | DG plus structural steel erection, hoists, precast concrete, safety nets, mast/tower cranes (erection) |
| **RI** | Intermediate rigging | RB plus rigging of cranes/conveyors/dredges, tilt-up slabs, demolition, dual-crane lifts |
| **RA** | Advanced rigging | RI plus gin poles and shear legs, flying foxes and cable-ways, guyed derricks and masts, suspended scaffolds (rigging) |

## Cranes, hoists and reach stackers

| Code | Class | Covers |
|---|---|---|
| **CT** | Tower crane | Jib or boom crane mounted on a tower structure |
| **CS** | Self-erecting tower crane | Tower crane transported as a unit; erection is an inherent function |
| **CD** | Derrick crane | Slewing strut-boom crane with boom pivoted at the base |
| **CP** | Portal boom crane | Boom or jib crane mounted on a portal frame running on rails/runway |
| **CB** | Bridge and gantry crane | Powered bridge/gantry crane controlled from a cabin or remote with ≥ 3 powered operations |
| **CV** | Vehicle loading crane | Crane mounted on a vehicle for loading/unloading, **≥ 10 tonne-metres** capacity |
| **CN** | Non-slewing mobile crane | Mobile crane with a non-slewing boom/jib, **> 3 tonnes** capacity |
| **C2** | Slewing mobile crane (≤ 20 t) | Slewing mobile crane up to 20 tonnes |
| **C6** | Slewing mobile crane (≤ 60 t) | Up to 60 tonnes |
| **C1** | Slewing mobile crane (≤ 100 t) | Up to 100 tonnes |
| **C0** | Slewing mobile crane (open) | Over 100 tonnes |
| **HM** | Materials hoist | Hoist for goods/materials only (externally cantilevered) |
| **HP** | Personnel and materials hoist | Hoist for people and materials in an enclosed car/structure |
| **WP** | Boom-type EWP | Elevating work platform with a **boom length ≥ 11 m** |
| **RS** | Reach stacker | Reach stacker > 3 tonnes capacity, handling freight containers |

> Slewing-crane classes are by **rated capacity**; a higher class covers lower capacities. **WP**
> only applies to boom-type EWPs of 11 m or more — smaller/scissor EWPs are not HRWL-licensed but
> still require training and competency.

## Forklifts

| Code | Class | Covers |
|---|---|---|
| **LF** | Forklift truck | Powered forklift with a mast and elevating load carriage (not order-picking) |
| **LO** | Order-picking forklift | Forklift where the operator's controls elevate **with** the load carriage |

## Pressure equipment

| Code | Class | Covers |
|---|---|---|
| **BB / BI / BA** | Boiler (basic / intermediate / advanced) | Operating boilers, by complexity (single vs multiple fuels, superheaters/economisers, etc.) |
| **ES** | Reciprocating steam engine | Steam engine with a piston diameter ≥ 250 mm |
| **TO** | Steam turbine | Steam turbine ≥ 500 kW, multi-stage or high-speed |

> Boiler class codes vary slightly between regulators' published lists (e.g. "standard" vs "basic")
> — confirm the exact code and scope with the relevant regulator.
